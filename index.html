Cara Bergabung

01

Buka Minecraft

Gunakan Minecraft Java atau Bedrock versi terbaru.

02

Multiplayer

Pilih menu Multiplayer lalu klik Add Server.

03

Input IP

Masukkan IP play.euodia.my.id di kolom Server Address.

04

Selamat Bermain!

Masuk ke server dan mulailah petualanganmu bersama kami.
